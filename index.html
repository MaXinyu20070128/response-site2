📐 二次方程求根器

ax² + bx + c = 0 (实数根 · a > 0, Δ ≥ 0)

🧮 标准形式: ax² + bx + c = 0 a > 0
💡 使用说明
• 点击上方按钮，按顺序输入 系数 a (a > 0)，b，c
• 程序假设 Δ = b² - 4ac ≥ 0 且 a > 0，但会进行合法性检查
• 计算结果将通过 alert() 弹窗显示，同时输出至 浏览器控制台 (Console)
• 多次点击按钮可反复测试不同系数，满足「多次测试」要求 ✅

⚠️ 测试中可能遇到的问题 (What problems do you encounter?)

  • ❌ 非数值输入：若用户输入字母、空字符串或取消输入，parseFloat 会得到 NaN，程序会提示“无效数字”并终止计算。
  • 📉 判别式 Δ < 0：虽然题目假设 Δ ≥ 0，但测试中输入如 a=1, b=1, c=1 会导致 Δ = -3，计算平方根得到 NaN。本程序会明确检测并提示“无实数根”。
  • 🧮 系数 a ≤ 0：题目假设 a > 0，但测试中若输入 a = 0 或负数，方程退化或非二次，程序会拒绝计算并提示“a 必须大于 0”。
  • 🔢 浮点数精度误差：例如 a=0.1, b=0.2, c=0.1 等，计算结果可能出现微小误差（如 -0.0000000001），显示时已做四舍五入处理，但控制台原始值可能带极小数。
  • 🔄 用户中途取消 Prompt：若任意一次弹窗点击“取消”，返回值为 null，程序会终止并提示“用户取消输入”，避免错误计算。
  • 📌 浏览器限制弹窗：某些浏览器可能阻止连续 prompt？但由于点击按钮主动触发，完全合法，无拦截风险。但若过快点击可能无影响。
  • 📐 忽略复数根：严格按照题目假设只处理 Δ≥0 情况，遇到负数 Δ 会明确告知用户问题，符合现实测试场景。
